RPG Game

  This is a project I made to get familiar with working inside the GB Studio game development environment, where you can walk your hometown, interacting with your neighbors and items. While very simple, this project made me appreciate the restrictions that were present when developing games for old school handheld devices. All source code is on my github.

Controls:  Start:[Enter]  Talk/Interact:[J]  Movement:Arrow Keys
